4. Sublime安装插件

Windows安装

1）打开Sublime，点击Preferences->Browse Packages…

2）关闭Sublime

3）删除下列图中Sublime Text 3文件夹

4）点击下载

5）解压 Sublime Text 3.zip，复制解压后的 Sublime Text 3 文件夹到刚才删除 Sublime Text 3的文件夹下，粘贴

6）打开 Sublime，正常使用

Mac安装

1）打开Sublime，点击下图所示，打开文件夹

2）关闭Sublime

3）删除下图中的Sublime Text 3 文件夹

4）点击下载

5）解压 Sublime Text 3.zip，复制解压后的 Sublime Text 3 文件夹到刚才删除 Sublime Text 3的文件夹下，粘贴（。。。/Library/Application Support）

6）打开Sublime，正常使用
